# AI-Assistant NSEDC
**Интеллектуальный агент для автоматизации полного цикла социально-экономических исследований от команды Барни**

## О проекте
Проект разработан в рамках VI Весенней школы «Информационные технологии и искусственный интеллект» (8-13 мая 2026).

#### Проблема:
1. **Галлюцинации ИИ**: Обычные нейросети (LLM) склонны «выдумывать» статистические показатели, названия датасетов и факты, опираясь на свою внутреннюю память, а не на реальные документы. В исследовательских задачах это делает их ответы недостоверными.
2. **Рутина**: Исследователи тратят до 70% времени на рутинный поиск верифицированных источников, согласование единиц измерения и борьбу с форматами данных (API, PDF, Parquet).

#### Решение:
AI-ассистент, который превращает запрос на естественном языке в готовый к анализу датасет, проходя через 7 этапов формализации и верификации

## Архитектура системы

Система построена на базе Hybrid Agentic Pipeline (гибридного агентного пайплайна), сочетающего строгость детерминированных алгоритмов и гибкость LLM.

#### Ключевые особенности:
- **State-Machine Logic**: Каждое исследование — это объект состояния (State), сохраняемый в БД.
- **RAG** (Retrieval-Augmented Generation): Семантический поиск по реестру НЦСЭД с использованием векторной базы FAISS.
- **SSE** Streaming: Интерактивное взаимодействие с пользователем в реальном времени через Server-Sent Events.
- **Trace Log** («След ассистента»): Полная прозрачность работы агента — пользователь видит каждый шаг и логику принятия решений.


## Технологический стек

- **Backend**: `FastAPI`, `SQLModel` (`SQLAlchemy` + `Pydantic`), `Uvicorn`
- **Database**: `SQLite` (для портативности и скорости)
- **ML**: 
    - LLM: `Qwen3-235b`
    - Vector Store: `FAISS`
    - Embeddings: intfloat/multilingual-e5-small
- **Frontend**: `React.js`

## Быстрый старт

**Предварительные требования**

    Python 3.12+
    Установленный менеджер пакетов uv

### Установка и запуск

1. **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-repo/barni_matmod_ai_agent.git
    cd barni_matmod_ai_agent
    ```
2. **Настройте окружение:**

    Создайте файл .env в корне проекта:
    ```bash
    AI_API_KEY=your_yandex_api_key
    AI_BASE_URL=https://llm.api.cloud.yandex.net/foundationModels/v1/api/openai/
    AI_MODEL=gpt://<folder_id>/qwen-max/latest
    JWT_SECRET=your_random_secret
    ```
   
3. **Установите зависимости и иницализируйте БД**:
    ```bash
    uv sync
    uv run python -m src.ml.rag build
    ```

4. **Запустите сервер:**
    ```bash
    uv run uvicorn src.interface.main:app --reload --port 8000
    ```
